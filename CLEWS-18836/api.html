API Reference¶

GroClient¶

class api.client.gro_client.GroClient(api_host, access_token)[source]¶

An extension of the Client class with extra convenience methods for some common operations.

Extra functionality includes: - Automatic conversion of units - Finding data series using entity names rather than ids - Exploration shortcuts for filling in partial selections - Saving data series in a data frame for repeated use

add_data_series(**kwargs)[source]¶

Search for entities matching the given names, find data series for the given combination, and add them to this objects list of series.

convert_unit(point, target_unit_id)[source]¶

Convert the data point from one unit to another unit.

If original or target unit is non-convertible, throw an error.

Parameters:
  • point (dict) – { value: float, unit_id: integer, ... }
  • target_unit_id (integer) –
Returns:

{ value: float, unit_id: integer, ... } unit_id is changed to the target, and value is converted to use the new unit_id. Other properties are unchanged.
Return type:

dict
get_data_points(**selections)[source]¶

Extend the Client’s get_data_points method to add unit conversion.

Parameters:selections (dict) – See lib.py get_data_points() for the base list of inputs This extended version may additionally include ‘unit_id’ which is the unit you wish to convert all points to.
Returns:Unchanged output format from lib.py get_data_points()
Return type:list of dicts
get_df()[source]¶

Get the content of all data series in a Pandas frame.

pick_random_data_series(selected_entities)[source]¶

Given a selection of tentities, pick a random available data series the given selection of entities.

pick_random_entities()[source]¶

Pick a random item that has some data associated with it, and a random metric and region pair for that item with data available.

print_one_data_series(data_series, filename)[source]¶

Output a data series to a CSV file.

search_for_entity(entity_type, keywords)[source]¶

Returns the first result of entity_type (which is items, metrics or regions) that matches the given keywords.
